# Retail-Sales-Analytics
Retail Sales Analytics Dashboard | Power BI, SQL, DAX, Data Modeling, KPI Reporting
#Grocery Store Retail Sales Analysis Report

## Objective
To conduct a comprehensive analysis of grocery store sales data, uncover product performance trends, evaluate the impact of promotions on sales, identify top-selling products, and deliver actionable insights through interactive Power BI dashboards to support data-driven decision-making

##Tools 
  - Tools Used: Power BI , DAX, Microsoft Excel,SQL, Power Query

- Techniques & Processes:
      - Built a star-schema data model connecting Sales, Products, Promotions, Stores, and Channels for efficient reporting and analysis.
      - Developed DAX measures and KPIs to track revenue growth, quantity variance, sales performance, and promotional effectiveness.
      - Transformed and cleaned data using Power Query to ensure accuracy, consistency, and reporting readiness.
      - Implemented interactive slicers, filters, drill-throughs, and conditional formatting to enhance data exploration.
      - Designed intuitive dashboards with a storytelling approach, enabling seamless navigation from summary KPIs to detailed insights.
      - Applied data visualization best practices to uncover trends, top-selling products, and key business drivers.
